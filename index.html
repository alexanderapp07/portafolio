Sobre mí

Hola, soy Alexander Aparicio, egresado de Ingeniería de Sistemas Informáticos con destrezas en programación, análisis de sistemas y gestión de bases de datos. Poseo conocimientos en desarrollo web e implementación de metodologías ágiles y en la automatización de procesos para alcanzar soluciones informáticas eficientes.

Proyectos

Sistema informático para el apoyo a la producción de serigrafía en Cygnus Investments S.A. DE C.V.

Sistema informático para optimizar y gestionar de manera eficiente el proceso de producción de serigrafía de camisetas. El sistema se divide en módulos interconectados que se encargan de cada etapa del proceso, permitiendo un seguimiento y control detallado.

Tecnologías utilizadas: PHP/Laravel y MariaDB

Diagramación, Diseño y desarrollo de sitio web de Asociación de Municipios Cayaguanca

El sitio web cuenta con una serie de módulos diseñados para garantizar la seguridad de la información, facilitar la administración eficiente y ofrecer una experiencia interactiva a los usuarios. A través de esta plataforma, los visitantes podrán acceder a las publicaciones, informarse sobre los eventos programados y los proyectos en curso.

Tecnologías utilizadas: PHP/Laravel, JavaScript y MariaDB

Backend para Sistema de Bolsa de Trabajo HireLink

Diseño de base de datos y desarrollo del backend para almacenar y gestionar los volúmenes de datos para la automatización de los procesos: gestión de empresas, oferta de puestos de trabajo y registro de candidatos. Además, la base de datos incluye soporte para la gestión de roles dinámicos, gestión de usuarios, y activación y bloqueo de usuarios.

Tecnologías utilizadas: Java/Spring Boot y PostgreSQL

Tiana's Boutique

Sistema informático para la optimización y gestión de pedidos para la venta de ropa y variedades online de la micro empresa bajo el nombre de Tiana's Boutique. Funcionalidades incluidas: gestión de usuarios, solicitud de pedido, entrega de productos, gestión de catálogos, gestión de categorias

Tecnologías utilizadas: PHP/Laravel, JavaScript y MariaDB
